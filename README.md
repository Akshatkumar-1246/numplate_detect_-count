This project is an Automatic Number Plate Recognition (ANPR) system built to detect and log vehicles in real time. A custom YOLOv8 model was trained on a number plate dataset sourced from Roboflow to accurately detect license plates in images and video frames. Once a plate is detected, the region is cropped and passed through an OCR engine to extract the alphanumeric text from the plate. The system then determines whether the vehicle is entering or exiting based on the detection context. All recognized plate numbers are automatically logged into an Excel sheet along with their IN/OUT status and timestamps. This makes it easy to maintain a structured record of vehicle movement without any manual entry. The entire pipeline — from detection to logging — is integrated into a user-friendly web application built with Streamlit. Users can upload images or video files, or use a live webcam feed for real-time detection directly from the browser. The project combines computer vision, OCR, and basic data management to solve a practical real-world problem in vehicle monitoring. It is intended as a deployable prototype for use cases like parking management, gated community access, or campus vehicle tracking.
